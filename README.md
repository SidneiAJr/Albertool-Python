## 📌 O que é isso?

Arquivos de dependência e esqueletos prontos pra você não perder tempo configurando o básico antes de codar em Python.  
Você recebe a organização. A lógica é sua.

---

## 📦 Arquivos de Dependência

Requirements prontos por especialidade — só instalar e sair usando.

| Arquivo | Uso |
|---------|-----|
| `requirements-basico.txt` | requests, dotenv, pydantic, MySQL |
| `requirements-pygame.txt` | Pygame, numpy, pymunk, Pillow |
| `requirements-flask.txt` | Flask, JWT, SQLAlchemy, MySQL/PostgreSQL |
| `requirements-fastapi.txt` | FastAPI, uvicorn, JWT, pydantic |
| `requirements-django.txt` | Django, DRF, JWT, celery, redis |
| `requirements-data.txt` | pandas, numpy, matplotlib, scikit-learn, jupyter |
| `requirements-exe.txt` | PyInstaller para gerar `.exe` |

```bash
pip install -r requirements-<especialidade>.txt
```

---

## 🦴 Esqueletos

Estruturas base prontas pra cada framework — só adicionar a lógica.

| Arquivo | O que tem |
|---------|-----------|
| `tkinter-basico.py` | Janela 800x600, label, input e botão |
| `pygame-basico.py` | Game loop, handle_events, update, draw |
| `flask-basico.py` | CRUD completo com rotas GET, POST, PUT, DELETE |
| `fastapi-basico.py` | CRUD async com rotas GET, POST, PUT, DELETE |
| `pandas-basico.py` | carregar, limpar, analisar, visualizar e exportar dados |
